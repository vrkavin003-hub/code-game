# 🎮 Code Runner: Neon Debug Quest

A fast-paced **coding-themed runner game** built with a modern frontend design, smooth animations, and a premium neon gaming UI.
Players enter a futuristic digital world where they must avoid bugs, collect points, and survive inside a glowing code universe.

## ⚙️ Installation and Setup

Clone the project:

```bash
git clone https://github.com/vrkavin003-hub/code-game.git
```

Go to the project folder:

```bash
cd code-game
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Build the project:

```bash
npm run build
```
